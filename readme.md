# tmpps.boardless.web

tmpps.boardless の Web サーバー

## Required

- node.js

#### Options

- vscode

## Usage

#### 始めに

`npm install`

#### 実行

`npm start`

#### プロダクションビルド

`npm run build`

#### テスト

`npm test`

#### CI 用テスト

`npm run test-ci`

#### API Mock サーバーの起動

`npm run mock`

#### electron package の作成

`npm run electron`

※ linux 環境で windows 用パッケージを作成する場合 `wine` がインストールされている必要がある
